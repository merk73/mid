# THE MIDGAS

Многостраничный художественный исследовательский архив: клиенты, аномалии, инциденты и историческая хронология.

> Содержание сайта является художественным вымыслом и архивной реконструкцией. Контактные, эзотерические и альтернативно-исторические версии не представлены как академически установленные факты.

## Локальный запуск

Сайт статический и не требует сборки:

```powershell
python -m http.server 43129 --directory web
```

После запуска откройте `http://127.0.0.1:43129/`.

## Структура

- `web/` — готовый сайт и все используемые ресурсы;
- `design/` — визуальная система и исследовательские заметки;
- `scripts/` — вспомогательные генераторы исходных материалов.

Основные страницы:

- `web/index.html` — главная;
- `web/registry.html` — реестры;
- `web/record.html` — карточки записей;
- `web/historical-archive.html` — историческая временная лента.
